# 🚌 Ficha Diária – Controle de Ponto

Sistema web para controle de ponto de motoristas de ônibus.  
Funciona em qualquer navegador — sem instalação.

## ▶ Acesso

**👉 https://SEU_USUARIO.github.io/ficha-diaria/**

### Login padrão (primeira vez)
| Campo | Valor |
|-------|-------|
| Usuário | `admin` |
| Senha | `admin123` |

⚠️ **Altere a senha após o primeiro acesso** usando o botão **🔐 Alterar Senha** no topo da tela.

## Funcionalidades

- 🔐 Login com senha (SHA-256, seguro)
- Cadastro ilimitado de motoristas
- Grade de marcações por quinzena (V 20h, V 11h, VE 20h, VE 11h, Falta, Atestado)
- Totais automáticos em tempo real
- Observações por motorista
- Exportação **Excel** (.xlsx) e **PDF** com preview
- Dados salvos no navegador (localStorage)
- Botão "Nova Quinzena" mantém motoristas
